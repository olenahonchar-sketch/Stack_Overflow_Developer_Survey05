# Stack Overflow Developer Survey 2025 Analysis

Project Overview:
A data analysis project examining the Stack Overflow 2025 survey responses from 49,191 developers. The analysis was conducted in Python using Pandas across 10 structured tasks to explore technology trends, compensation, and work habits.

Key Tasks & Results:
1.  Total Respondents: 49,191 developers.
2.  Response Completeness: 0% of respondents answered all 126 core questions, typical for large optional surveys.
3.  Work Experience Stats: Average experience is 7.8 years (median: 6 years).
4.  Remote Work: 35.9% (17,663 developers) work remotely or in a hybrid model.
5.  Python Popularity: Used by 58.3% of developers who listed their languages (18,410 individuals).
6.  Learning Pathways: 32.7% (10,973 developers) used online courses or certifications to learn.
7.  Python Salaries by Country: Significant pay gaps exist. Top countries by number of Python developers and their median salary: USA (3,126 devs, $150K), Germany (1,188 devs, $81K), UK (750 devs, $95K), France (588 devs, $60K), India (543 devs, $17K).
8.  Top Earners' Education: After filtering unrealistic salary outliers, the highest salaries were held by those with Master's and Bachelor's degrees.
9.  Python Use by Age: Strong negative correlation with age. Usage is 71.8% among 18-24 year-olds but drops to 43.5% among developers aged 65+.
10. Top Industries for High-Paid Remote Workers: For remote workers earning above $120,596 (75th percentile), the dominant industry is Software Development (42.9% or 1,820 workers).

Technologies & Skills: Python, Pandas. Skills demonstrated: data cleaning, handling missing values and outliers, statistical analysis (mean, median, percentiles), data grouping, aggregation, and filtering.

Огляд проєкту:
Проєкт з аналізу даних, що досліджує відповіді 49,191 розробника з опитування Stack Overflow 2025. Аналіз виконувався на Python з використанням Pandas у 10 структурованих завданнях для вивчення тенденцій у технологіях, компенсаціях та звичках щодо праці.

Основні завдання та результати:
1.  Загальна кількість респондентів: 49,191 розробник.
2.  Повнота відповідей: 0% респондентів відповіли на всі 126 основних запитань, що типовий результат для великих опціональних опитувань.
3.  Статистика досвіду: Середній досвід роботи — 7.8 років (медіана: 6 років).
4.  Віддалена робота: 35.9% (17,663 розробники) працюють віддалено або в гібридному режимі.
5.  Популярність Python: Використовується 58.3% розробників, які вказали свої мови (18,410 осіб).
6.  Шляхи навчання: 32.7% (10,973 розробників) вчилися через онлайн-курси чи сертифікати.
7.  Зарплати Python-розробників за країнами: Існують значні розриви в оплаті. Топ-5 країн за кількістю Python-розробників та їх медіанна зарплата: США (3,126 розр., $150К), Німеччина (1,188 розр., $81К), Велика Британія (750 розр., $95К), Франція (588 розр., $60К), Індія (543 розр., $17К).
8.  Освіта найвищооплачуваніших: Після фільтрації нереалістичних викидів у зарплатах, найвищі компенсації отримували особи зі ступенем магістра та бакалавра.
9.  Використання Python за віком: Чітка негативна кореляція з віком. Використання серед розробників 18-24 років становить 71.8%, але знижується до 43.5% серед тих, кому 65+.
10. Топ індустрії для високооплачуваних віддалених працівників: Серед віддалених працівників із зарплатою вище $120,596 (75-й процентиль), домінує індустрія розробки програмного забезпечення (42.9% або 1,820 працівників).

Технології та навички: Python, Pandas. Продемонстровані навички: очищення даних, робота з пропущеними значеннями та викидами, статистичний аналіз (середнє, медіана, процентилі), групування, агрегація та фільтрація даних.
